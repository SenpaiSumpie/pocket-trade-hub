  <action>
    1. Install dependencies: `cd apps/mobile && npx expo install expo-image @shopify/flash-list`

    2. Create `apps/mobile/src/stores/cards.ts` (Zustand store):
       - State: selectedSetId (string | null), searchQuery (string), activeFilters ({ set?: string, rarity?: string, type?: string }), isSearchMode (boolean)
       - Actions: setSelectedSet, setSearchQuery, setFilter, removeFilter, clearFilters, toggleSearchMode

    3. Create `apps/mobile/src/hooks/useCards.ts`:
       - `useSets()` — fetches GET /sets on mount, returns { sets, loading, error }
       - `useCardsBySet(setId, limit=50)` — fetches GET /sets/:id/cards with pagination, returns { cards, loading, hasMore, loadMore }
       - `useCardSearch(query, filters)` — fetches GET /cards/search with 300ms debounce (useEffect + setTimeout pattern from RESEARCH.md), only fires when query >= 2 chars OR filters active, returns { results, loading, total }
       - All hooks use apiFetch from useApi.ts

    4. Create `apps/mobile/src/components/cards/RarityBadge.tsx`:
       - Renders Pokemon TCG Pocket rarity symbols using Unicode:
         - diamond1-4: repeat diamond character (U+2666) 1-4 times, light blue color
         - star1-3: repeat star character (U+2605) 1-3 times, gold color (#f0c040)
         - crown: crown character (U+1F451), rainbow/special color
       - Compact layout, used in both thumbnail and detail view

    5. Create `apps/mobile/src/components/cards/CardThumbnail.tsx`:
       - Pressable card cell for 3-column grid
       - expo-image with blurhash placeholder, recyclingKey={card.id}, contentFit="cover", aspect ratio 1/1.4
       - Below image: card name (truncated), RarityBadge, type icon (small colored dot or Ionicon)
       - onPress callback prop to open detail
       - When in search mode: small set badge overlay in corner showing set name

    6. Create `apps/mobile/src/components/cards/SetPicker.tsx`:
       - Horizontal ScrollView (or FlatList horizontal) of set chips
       - Each chip shows set name, highlighted when selected (gold accent border/background)
       - Tapping selects set, calls setSelectedSet in store
       - First set auto-selected on load

    7. Create `apps/mobile/src/components/cards/SearchBar.tsx`:
       - TextInput with search icon (Ionicons search), persistent above set picker
       - Dark themed (card background color), rounded corners
       - Controlled by cards store searchQuery
       - Clear button (X icon) when text present
       - onChangeText updates store, which triggers debounced search via useCards hook

    8. Create `apps/mobile/src/components/cards/FilterChips.tsx`:
       - Row of filter chips below search bar: Set, Rarity, Type
       - Tapping a chip opens a bottom sheet or modal picker with options
       - Set options: all set names from useSets()
       - Rarity options: diamond1-4, star1-3, crown (show rarity symbols)
       - Type options: fire, water, grass, lightning, psychic, fighting, darkness, metal, dragon, colorless, normal
       - Active filters shown as dismissible chips with X button
       - Only visible when search is active (searchQuery has text)

    9. Create `apps/mobile/src/components/cards/CardGrid.tsx`:
       - FlashList with numColumns={3}, estimatedItemSize={180}
       - Renders CardThumbnail for each card
       - Loading skeleton placeholders while fetching (3x3 grid of grey rounded rectangles with pulse animation)
       - Empty state for no results: "No cards found" with suggestion text
       - Infinite scroll: onEndReached calls loadMore when hasMore is true
       - Receives cards array and loading state as props

    10. Create `apps/mobile/src/components/cards/CardDetailModal.tsx`:
        - Full-screen modal presentation (use React Native Modal or Expo Router modal)
        - Horizontal FlatList/PagerView for swipe between cards in the set
        - Large card image at top (expo-image, full width, aspect ratio 1/1.4)
        - Below image, scrollable stats section:
          - Card name (large), set name, card number (e.g. "042/286")
          - RarityBadge (larger size), type
          - HP (if pokemon)
          - Attacks: list each with name, damage, energy cost icons, description
          - Weakness, Resistance, Retreat cost
        - Two action buttons at bottom:
          - "Add to Collection" — disabled, opacity 0.5, text "Coming in next update"
          - "Add to Wanted" — disabled, opacity 0.5, text "Coming in next update"
        - Close button (X) in top-right corner
        - Dark background matching app theme
