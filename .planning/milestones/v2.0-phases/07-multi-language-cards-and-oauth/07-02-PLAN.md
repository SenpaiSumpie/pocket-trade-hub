Modify `apps/api/src/services/card.service.ts`:

1. Import `cardTranslations` from schema
2. Update `searchCards`:
   - When `params.language` is provided, LEFT JOIN `cardTranslations` on (cardId, language)
   - Replace card.name/imageUrl with translation.name/imageUrl when available
   - When `params.q` is provided WITH language, search against `cardTranslations.name` instead of `cards.name`

   - Return array of `{ language, name, imageUrl, attacks }`

   - Add optional `language` parameter
   - When provided, join with cardTranslations to get translated name/imageUrl
   - Return base card data merged with translation data
