  <action>
1. Install dependencies:
   - `cd apps/api && pnpm add cheerio axios`
   - `cd apps/mobile && pnpm add react-native-draggable-flatlist`

2. Add 4 new tables to apps/api/src/db/schema.ts (append after existing tables):

   **deckMeta table:**
   - id: text PK
   - name: varchar(200) notNull
   - winRate: integer (basis points, e.g. 5022 = 50.22%)
   - usageRate: integer (basis points)
   - playCount: integer
   - matchRecord: varchar(50) -- "W-L-D" format
   - cards: jsonb -- array of card images/names from scrape
   - matchups: jsonb -- { good: [...], bad: [...] }
   - tournamentResults: jsonb
   - scrapedAt: timestamp notNull
   - createdAt: timestamp defaultNow notNull
   - updatedAt: timestamp defaultNow notNull
   - Indexes: win_rate, usage_rate

   **tradeSuggestions table:**
   - id: text PK
   - userId: text FK -> users.id notNull
   - giveCardId: text FK -> cards.id notNull
   - getCardId: text FK -> cards.id notNull
   - score: integer notNull
   - reasoning: text notNull
   - computedAt: timestamp defaultNow notNull
   - Index: user_id

   **tierLists table:**
   - id: text PK
   - userId: text FK -> users.id notNull
   - title: varchar(100) notNull
   - description: text (nullable)
   - tiers: jsonb notNull -- { S: [...deckIds], A: [...], B: [...], C: [...], D: [...] }
   - isOfficial: boolean default(false) notNull
   - upvoteCount: integer default(0) notNull
   - createdAt: timestamp defaultNow notNull
   - updatedAt: timestamp defaultNow notNull
   - Indexes: user_id, is_official, upvote_count

   **tierListVotes table:**
   - id: text PK
   - tierListId: text FK -> tierLists.id with onDelete cascade notNull
   - userId: text FK -> users.id notNull
   - createdAt: timestamp defaultNow notNull
   - Unique index: (tierListId, userId)

3. Create shared Zod schemas:

   **packages/shared/src/schemas/meta.ts:**
   - deckMetaSchema: { id, name, winRate, usageRate, playCount, matchRecord, cards, matchups, tournamentResults, scrapedAt }
   - deckMetaResponseSchema: { decks: array of deckMetaSchema, isPremium: boolean }
   - Export inferred types: DeckMeta, DeckMetaResponse

   **packages/shared/src/schemas/suggestion.ts:**
   - tradeSuggestionSchema: { id, giveCardId, getCardId, giveCard (name, rarity, imageUrl), getCard (name, rarity, imageUrl), score, reasoning, computedAt }
   - suggestionsResponseSchema: { suggestions: array, isPremium: boolean }
   - Export inferred types: TradeSuggestion, SuggestionsResponse

   **packages/shared/src/schemas/tierlist.ts:**
   - tierEntrySchema: { deckId, deckName }
   - tiersSchema: { S: array, A: array, B: array, C: array, D: array } of tierEntrySchema
   - tierListSchema: { id, userId, title, description, tiers, isOfficial, upvoteCount, userVoted (optional boolean), createdAt, updatedAt }
   - createTierListSchema: { title, description (optional), tiers }
   - tierListResponseSchema: { tierLists: array, total: number }
   - Export inferred types

4. Re-export all 3 new schema files from packages/shared/src/index.ts.

5. Run `cd apps/api && npx drizzle-kit push` to apply schema changes to dev DB.

1. Create apps/api/src/services/meta-scraper.service.ts:
   - Import cheerio and axios
   - export async function scrapeDeckMeta(): Promise<ScrapedDeck[]>
   - Fetch https://play.limitlesstcg.com/decks?game=pocket with User-Agent header ("PocketTradeHub/1.0")
   - Parse HTML table rows using cheerio: extract deck name, play count, usage %, win rate from table cells
   - Return array of parsed deck objects with all available fields
   - Handle errors gracefully: if scrape returns 0 decks or throws, log error and return empty array
   - Add retry with exponential backoff (1 attempt, 3s delay) for transient network failures
   - Do NOT scrape on every user request -- this is called from BullMQ worker only

2. Create apps/api/src/services/meta.service.ts:
   - import { db } from pattern, use drizzle queries on deckMeta table
   - export async function upsertDeckMeta(db, decks: ScrapedDeck[]): upsert all scraped decks into deckMeta table using nanoid for new IDs, update on conflict by name
   - export async function getDeckMeta(db, isPremium: boolean): return all decks sorted by winRate desc. If !isPremium, strip matchups, tournamentResults, and limit cards to top 3 (per CONTEXT.md free/premium split)
   - export async function getDeckDetail(db, deckId: string, isPremium: boolean): return single deck with full or limited data based on premium status
   - export async function getScrapedAt(db): return most recent scrapedAt timestamp for "last updated" display
   - All functions follow existing service pattern (receive db instance as first param)
