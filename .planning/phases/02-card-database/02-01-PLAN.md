    1. Create `apps/api/src/db/seeds/seed-cards.ts`:
       - Fetches Pokemon TCG Pocket series from TCGdex API: `https://api.tcgdex.net/v2/en/series/tcgp`
       - For each set in the series, fetches full set data: `https://api.tcgdex.net/v2/en/sets/{setId}`
       - Transforms TCGdex data to match our Drizzle schema:
         - Set: id, name, series extracted from set data, cardCount from cards array length, releaseDate, imageUrl from TCGdex
         - Card: id = `{setId}-{localId}`, map fields (name, rarity mapping from TCGdex rarity to our enum values, type, category, hp, stage, imageUrl from `https://assets.tcgdex.net/en/tcgp/{setId}/{localId}`, attacks as jsonb, weakness, resistance, retreatCost, illustrator, cardNumber)
       - Rarity mapping from TCGdex format to our enum: map common/uncommon/rare patterns to diamond1-4/star1-3/crown based on TCGdex rarity field
       - Inserts in batches of 50 within a database transaction per set
       - Logs progress: "Seeded [SetName]: [N] cards"
       - Handles gracefully if set already exists (skip with warning)
       - Add 500ms delay between API calls to be respectful of TCGdex rate limits
    2. Add npm script to `apps/api/package.json`: `"db:seed": "tsx src/db/seeds/seed-cards.ts"`
    3. Script should be runnable standalone: `cd apps/api && pnpm db:seed`
    4. Include error handling: if TCGdex API is unreachable, log error and exit gracefully with instructions
  </action>
