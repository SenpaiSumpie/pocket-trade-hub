## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| 2xs | 2px | Hairline gaps, border-adjacent micro-spacing |
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, input inner padding |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, list item vertical rhythm |
| xl | 32px | Layout gaps between major sections |
| xxl | 48px | Major section breaks, screen-level top padding |
| 3xl | 64px | Page-level spacing, modal vertical offset |

Exceptions: 2px (2xs) is not a multiple of 4 but is included per D-05 for hairline/micro-spacing needs. Mobile uses raw pixel values; web CSS generation divides by 16 for rem units.

### Accent Reserved For

The gold accent (`#f0c040`) is reserved exclusively for:
- Primary action buttons (CTA buttons)
- Active tab indicator
- Selected/active navigation states
- Premium/gold feature highlights
- Pull-to-refresh tint (Phase 16)
- Star rarity indicator (shared value)

Never use accent for: borders, backgrounds, body text, secondary actions, disabled states, or decorative elements.

### Warning vs Gold Differentiation

Warning (`#e67e22`, orange) is visually distinct from accent (`#f0c040`, gold). Warning skews toward red-orange while accent is a warm yellow-gold. Tested differentiation: 40-degree hue separation on the color wheel.

The hardcoded value audit (DS-05) produces a structured tracking list with these column headers:
- **File** — relative path from repo root
- **Line** — line number of the hardcoded value

## Cross-Platform Token Contract

This section is specific to Phase 13's cross-platform requirement (DS-04).

### Mobile Token Consumption

- Tokens imported as TypeScript objects from `@pocket-trade-hub/shared`
- Used directly in `StyleSheet.create()` calls
- Spacing values used as raw pixel numbers
- Backward-compatible shim at `apps/mobile/src/constants/theme.ts` re-exports new tokens with old property names

### Web Token Consumption

- `tokens.css` generated by Node build script from TS token source
- Imported in `globals.css` replacing current hand-written `@theme` block
- Spacing values converted to rem (divided by 16)
- `apps/web/src/lib/constants.ts` THEME object replaced with imports from shared tokens
- Generated file added to `.gitignore` — built in Turborepo pipeline

### Shim API Contract (DS-03)

The shim must export these exact shapes to avoid breaking the 44 consuming files:

```typescript
// colors: exact property names from current theme.ts
export const colors: {
  background: string;
  surface: string;
  surfaceLight: string;
  primary: string;       // maps to accent
  primaryDark: string;   // maps to accentDark
  text: string;          // maps to onSurface
  textSecondary: string; // maps to onSurfaceSecondary
  textMuted: string;     // maps to onSurfaceMuted
  error: string;
  success: string;
  border: string;
  tabBar: string;
  inputBackground: string;
};

// spacing: exact property names
export const spacing: { xs: number; sm: number; md: number; lg: number; xl: number; xxl: number; };

// borderRadius: exact property names
export const borderRadius: { sm: number; md: number; lg: number; xl: number; full: number; };

// typography: exact shapes with fontSize, fontWeight, color
export const typography: {
  heading: { fontSize: number; fontWeight: string; color: string; };
```
