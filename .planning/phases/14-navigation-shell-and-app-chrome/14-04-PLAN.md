Phosphor usage pattern:
```typescript
// OLD (Ionicons):
import { Ionicons } from '@expo/vector-icons';
<Ionicons name="checkmark" size={20} color="#fff" />
<Ionicons name={focused ? 'heart' : 'heart-outline'} size={20} color={color} />

// NEW (Phosphor):
import { Check, Heart } from 'phosphor-react-native';
<Check size={20} color="#fff" weight="regular" />
<Heart size={20} color={color} weight={focused ? 'fill' : 'regular'} />
```

Key differences:
- Ionicons uses string name prop; Phosphor uses named component imports
- Ionicons outline/fill variants use different name strings; Phosphor uses weight prop ('regular', 'fill', 'bold', 'light', 'thin', 'duotone')
- Ionicons size defaults vary; Phosphor defaults to 24 (set size explicitly)
