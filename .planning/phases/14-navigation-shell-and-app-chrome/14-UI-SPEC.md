# Phase 14 -- UI Design Contract

> Visual and interaction contract for the navigation shell and app chrome. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (React Native StyleSheet.create with shared token package) |
| Preset | not applicable |
| Component library | none (custom components consuming @pocket-trade-hub/shared tokens) |
| Icon library | phosphor-react-native 3.0.3, weight prop for regular/fill variants |

## Spacing Scale

Declared values from `packages/shared/src/tokens/spacing.ts` (all multiples of 4 except 2xs):

| Token | Value | Usage |
|-------|-------|-------|
| 2xs | 2px | Micro adjustments (badge offset, border gaps) |
| xs | 4px | Icon-to-label gaps, inline padding |
| sm | 8px | Compact element spacing, tab bar icon-label gap |
| md | 16px | Default element spacing, tab bar horizontal padding |
| lg | 24px | Section padding, header content padding |
| xl | 32px | Layout gaps, header expanded height content area |
| xxl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Tab bar height: 60px (not a token -- component-specific dimension for 44px touch target + 16px bottom safe area allowance)
- Collapsible header max height: 120px (component-specific, not a spacing token)
- Collapsible header min height: 50px (collapsed state showing title + bell only)
- Touch target minimum: 44px (accessibility standard, applied to each tab hit area)

| Element | Typography Role | Additional Styles |
|---------|----------------|-------------------|
| Tab bar labels (active) | Caption (13px) | color: accent (#f0c040), fontWeight 600 |

## Color

All values from `packages/shared/src/tokens/colors.ts` backed by `primitives.ts`.

| Role | Token | Hex Value | Usage |
|------|-------|-----------|-------|
| Dominant (60%) | background | #0c0c18 | Full-screen background behind all content |
| Secondary (30%) | tabBar | #111122 | Tab bar background, header background |
| Secondary (30%) | surface | #1a1a2e | Card surfaces, elevated containers |
| Accent (10%) | accent | #f0c040 | See reserved list below |
| Border | border | #2a2a45 | Tab bar top border (1px), header bottom separator |
| Text primary | onSurface | #ffffff | Active header titles, primary text |
| Text secondary | onSurfaceSecondary | #a0a0b8 | Secondary labels |
| Text muted | onSurfaceMuted | #6c6c80 | Inactive tab labels, placeholder text |
| Destructive | error | #e74c3c | Destructive actions only (not used in this phase's chrome) |

### Accent Reserved For (Phase 14)

The gold accent (#f0c040) is reserved exclusively for:
1. Active tab icon tint color
2. Active tab label color
3. Sliding pill indicator background (at 15% opacity: rgba(240, 192, 64, 0.15))
4. Gold dot notification badge on Trades tab
5. Notification bell icon when unread notifications exist

Accent is NOT used for: inactive tab elements, header backgrounds, borders, or general UI chrome.

| Property | Value |
|----------|-------|

| Damping | 20 |
| Stiffness | 200 |

## Interaction States

### Tab Bar States

| State | Visual Treatment |
|-------|-----------------|
| Default (inactive) | Regular-weight icon (#6c6c80) + label (#6c6c80), no pill behind |
| Active | Fill-weight icon (#f0c040) + label (#f0c040), gold pill behind (15% opacity) |
| Pressed (touchable feedback) | Slight opacity reduction (0.7) on the tab area during press |
| Trades with notifications | Gold dot badge visible on Trades icon (only when pending > 0) |

### Collapsible Header States

| State | Visual Treatment |
|-------|-----------------|
| Fully expanded | Title (28px) + bell + search/filters visible, 120px height |
| Collapsing (mid-scroll) | Continuous interpolation, search fading out |
| Fully collapsed | Title (20px) + bell only, 50px height, bottom border appears |
| Re-expanding (scroll up) | Spring animation back to expanded, search fading in |
