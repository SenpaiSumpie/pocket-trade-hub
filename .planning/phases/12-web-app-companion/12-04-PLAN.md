<interfaces>
API endpoints (from existing routes):
- GET /posts?type=offering|seeking&query=&setId=&rarity=&language=&page=&limit= -> { posts: MarketPost[], total }
- POST /posts -> create post { type, cards: [{ cardId, language }] }
- GET /posts/:id -> MarketPost with cards and poster info
- DELETE /posts/:id -> delete own post
- GET /proposals -> { sent: Proposal[], received: Proposal[] }
- POST /proposals -> create proposal { postId, offeredCards, requestedCards, message? }
- PUT /proposals/:id/accept -> accept
- PUT /proposals/:id/reject -> reject
- POST /proposals/:id/counter -> counter-offer { offeredCards, requestedCards, message? }
