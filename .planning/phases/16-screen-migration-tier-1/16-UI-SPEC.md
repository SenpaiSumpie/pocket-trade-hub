## Spacing Scale

All values sourced from `packages/shared/src/tokens/spacing.ts`. Multiples of 4 throughout.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gaps, inline chip padding |
| sm | 8px | Internal card padding, compact list row gaps |
| md | 16px | Default element spacing, card padding, input horizontal padding |
| lg | 24px | Section padding, card vertical padding |
| xl | 32px | Layout gaps between major sections |
| xxl | 48px | Major section breaks (Home tab between dashboard cards) |
| 3xl | 64px | Page-level top spacing (below collapsible header) |

Exceptions:
- Touch targets for all tappable elements: minimum 44px height (accessibility floor, not a spacing token)
- Toast overlay bottom inset: 16px above tab bar (tab bar height + 16px margin)
- Pull-to-refresh tint: not a spacing value ŌĆö color-only config (`tintColor="#f0c040"`)

Source: `packages/shared/src/tokens/spacing.ts` (confirmed values)

## Typography

All values sourced from `packages/shared/src/tokens/typography.ts`. Inter font family throughout.

Two weights only: 400 (Regular) and 700 (Bold).

| Role | Size | Weight | Line Height | Token Preset |
|------|------|--------|-------------|--------------|
| Heading | 28px | 700 (Bold) | 34px (1.21) | `typography.heading` |
| Subheading | 20px | 700 (Bold) | 26px (1.30) | `typography.subheading` |
| Body | 16px | 400 (Regular) | 22px (1.375) | `typography.body` |
| Label | 13px | 400 (Regular) | 18px (1.38) | `typography.label` |

Usage rules:
- **Heading**: Screen titles, section headers, EmptyState main title
- **Subheading**: Card section titles, segment labels, modal sheet titles, EmptyState section title
- **Body**: List item content, description text, toast messages, input values, button text (all sizes)
- **Label**: Secondary metadata, timestamps, card count labels, status sub-text, badge text, input labels, filter chip text

Text component default color: `colors.onSurface` (`#ffffff`).
Secondary/muted text: `colors.onSurfaceSecondary` (`#a0a0b8`) or `colors.onSurfaceMuted` (`#6c6c80`).

Source: `packages/shared/src/tokens/typography.ts` (confirmed values), CONTEXT.md D-04

## Color

All values sourced from `packages/shared/src/tokens/colors.ts` and `packages/shared/src/tokens/primitives.ts`.

### 60/30/10 Distribution

| Role | Value | Token | Usage |
|------|-------|-------|-------|
| Dominant (60%) | `#0c0c18` | `palette.gray[950]` | App background (`colors.background`) ŌĆö fills all screen backgrounds |
| Secondary (30%) | `#1a1a2e` | `palette.gray[800]` | Card surfaces (`colors.surface`), input backgrounds, toast card background |
| Accent (10%) | `#f0c040` | `palette.gold[500]` | `colors.accent` ŌĆö reserved for specific elements listed below |
| Destructive | `#e74c3c` | `palette.red[500]` | `colors.error` ŌĆö destructive button variant, error toast accent bar, error badge |

### Accent Reserved For (explicit list)

The gold accent (`#f0c040`) is used ONLY for:
1. Primary Button fill background
2. Input focus ring border
3. Pull-to-refresh tint (`tintColor` / `colors` prop on RefreshControl)
4. Star rarity shimmer overlay gradient (`#f0c040` at 15% opacity sweep on CardThumbnail)
5. Custom tab bar active pill indicator (Phase 14 ŌĆö already implemented)
6. Active segment tab underline / selected state indicators

The gold accent is NOT used for: body text, card borders, dividers, icon fills in idle state, list backgrounds.

### Supplementary Semantic Colors

| Token | Value | Usage |
|-------|-------|-------|
| `colors.success` | `#2ecc71` | Success toast accent bar + CheckCircle icon, success badge |
| `colors.warning` | `#e67e22` | Warning toast accent bar + Warning icon, warning badge |
| `colors.error` | `#e74c3c` | Error toast accent bar + XCircle icon, error badge, destructive Button |
| `colors.surfaceLight` | `#252540` | Elevated surface for segment toggle backgrounds, filter chip default state |
| `colors.border` | `#2a2a45` | Divider color, card border (when used), input default border |
| `colors.rarityDiamond` | `#7ec8e3` | Diamond rarity badge ŌĆö no glow/shimmer effect on card thumbnail |
| `colors.rarityStar` | `#f0c040` | Star rarity badge + shimmer overlay gradient on CardThumbnail |
| `colors.rarityCrown` | `#e8b4f8` | Crown rarity badge + pulsing glow border on CardThumbnail |

### Info Color (Toast)

The info toast uses blue. There is no `colors.info` token ŌĆö use hardcoded `#3498db` (matching `tierD` in colors.ts) with a Phosphor `Info` icon. This is the only hardcoded hex allowed in Phase 16.

Source: `packages/shared/src/tokens/colors.ts`, `packages/shared/src/tokens/primitives.ts`, CONTEXT.md D-17

## Skeleton Loading States

All skeletons use Phase 15 shimmer primitives: `Shimmer`, `ShimmerBox`, `ShimmerCircle`, `ShimmerText`.
Shimmer sweep color: white-to-transparent gradient (default from Phase 15 ŌĆö no override needed here).
Shimmer timing: `TIMING_SHIMMER` (1200ms linear ŌĆö from `springs.ts`).

### CardGridSkeleton (D-09)

- 9-item grid (3 columns ├Ś 3 rows), matching first visible page of CardGrid
- Each cell: ShimmerBox at card thumbnail aspect ratio (approximately 100px wide ├Ś 140px tall)
- Column gap: `spacing.sm` (8px)
- Row gap: `spacing.sm` (8px)

### PostListSkeleton (D-09)

- 3 rows, each row: ShimmerCircle (40px diameter) + two ShimmerText lines (80% width + 50% width)
- Row height: 72px
- Row padding: `spacing.md` (16px) horizontal
- Separator: Divider between rows

### ProposalListSkeleton (D-09)

- 3 rows, each row: ShimmerCircle (40px) + ShimmerBox (card preview, 32px ├Ś 44px) + two ShimmerText lines
- Row height: 88px
- Row padding: `spacing.md` (16px) horizontal
- Separator: Divider between rows

## Toast/Snackbar System

Custom Zustand-based system. Does NOT replace existing `react-native-toast-message` usage (matchNotification type) ŌĆö coexists in a new namespace. (D-15 through D-19, RESEARCH.md discovery)

| Property | Value |
|----------|-------|
| Position | Bottom of screen, above tab bar |
| Bottom inset | Tab bar height + `spacing.md` (16px) |
| Width | Screen width minus `spacing.md` ├Ś 2 (16px horizontal margin each side) |
| Background | `colors.surface` (#1a1a2e) |
| Border radius | `borderRadius.lg` (16px) |
| Elevation | `elevation.medium` |
| Left accent bar | 4px wide, full height, rounded-left corners, variant color |
| Auto-dismiss | 3000ms |
| Dismiss animation | fade out + translate Y +8px (slide down), 200ms timing |
| Enter animation | fade in + translate Y from +16px to 0, 200ms timing |
| Swipe to dismiss | swipe-down gesture, minimum 40px drag to trigger |
| Queue behavior | One at a time ŌĆö next toast shows after current dismisses |

### Variant Colors

| Variant | Accent Bar Color | Icon | Icon Color |
|---------|-----------------|------|-----------|
| success | `colors.success` (#2ecc71) | `CheckCircle` (Phosphor, filled) | `colors.success` |

## Animated Press Behavior

All tappable elements use `useAnimatedPress` from Phase 15. (D-20, D-21)

| Element | Press scale | Haptic |
|---------|-------------|--------|
| Button (all variants) | 0.97 | `Haptics.impactAsync(ImpactFeedbackStyle.Light)` |
| Card (with onPress) | 0.97 | `Haptics.impactAsync(ImpactFeedbackStyle.Light)` |
| MyPostCard, ProposalCard, MatchCard | 0.97 | `Haptics.impactAsync(ImpactFeedbackStyle.Light)` |
| Filter chips / segment buttons | 0.95 | `Haptics.impactAsync(ImpactFeedbackStyle.Light)` |
| EmptyState CTA button | via Button primitive | via Button primitive |

Spring preset for press: `SPRING_PRESS` (`{ damping: 15, stiffness: 300, mass: 0.8 }`) from `springs.ts`.

## Staggered List Animation

`useStaggeredList` from Phase 15. Applied on initial mount ŌĆö does not re-trigger on scroll. (D-31, D-37, D-41)

| Screen | Applied To | Stagger delay |
|--------|-----------|---------------|
| Home tab | Dashboard sections (setup checklist, collection summary, smart trades, analytics cards) | `TIMING_STAGGER_DELAY` (50ms) |
| Cards tab | Card grid items on initial load | `TIMING_STAGGER_DELAY` (50ms) |
| Trades tab | Post list items, proposal list items | `TIMING_STAGGER_DELAY` (50ms) |

Spring preset: `SPRING_CARD_APPEAR` (`{ damping: 12, stiffness: 120, mass: 1 }`).
Maximum staggered items: `MAX_STAGGER_ITEMS` (15) ŌĆö items beyond 15 appear without stagger.
Enter animation: fade from 0 to 1 + translate Y from 16px to 0.
