   - State: `currentLanguage: string` (default 'en'), `isLoading: boolean`

     - `setLanguage(lang: string)`: Calls i18n.changeLanguage(lang), updates store, PATCHes /users/me with { uiLanguage: lang } via apiFetch. Optimistic update — change UI immediately, sync to server in background.
