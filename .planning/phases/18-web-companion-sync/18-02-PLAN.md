**Toast store (D-17):** Create `apps/web/src/stores/toast.ts`:
- Import `create` from `zustand`
- Export `ToastVariant` type: `'success' | 'error' | 'info' | 'warning'`
- Export `ToastItem` interface: `{ id: string; variant: ToastVariant; message: string }`
- `ToastState` interface: `{ queue: ToastItem[]; show: (variant: ToastVariant, message: string) => void; dismiss: (id: string) => void }`
- Export `useToastStore`:
  - `show`: caps queue at 4 items: `set((s) => ({ queue: [...s.queue.slice(-3), { id: Date.now().toString(), variant, message }] }))`
  - `dismiss`: filters by id: `set((s) => ({ queue: s.queue.filter((t) => t.id !== id) }))`
