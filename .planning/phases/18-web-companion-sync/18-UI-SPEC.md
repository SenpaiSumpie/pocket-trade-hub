## Typography

Token source: `apps/web/src/app/tokens.css`. Four declared sizes from shared token package.

| Role | Size | CSS Var | Weight | Line Height | Usage |
|------|------|---------|--------|-------------|-------|
| Heading | 28px (1.75rem) | `--font-size-heading` | 700 (`--font-weight-bold`) | 34px (2.125rem) | Page titles (h1), modal titles |
| Subheading | 20px (1.25rem) | `--font-size-subheading` | 700 (`--font-weight-bold`) | 26px (1.625rem) | Section headers (h2), card titles |
| Body | 16px (1rem) | `--font-size-body` | 400 (`--font-weight-regular`) | 22px (1.375rem) | Default text, descriptions, form content |
| Label | 14px (0.875rem) | `--font-size-label` | 400 (`--font-weight-regular`) | 20px (1.25rem) | Button labels, nav items, form labels, badge text, helper text, timestamps, metadata |

Two weights only: 400 (regular — body, label, secondary text) and 700 (bold — headings, subheadings). Per D-18, no web Text primitive — Tailwind utility classes handle typography directly using `text-[var(--font-size-X)]` or Tailwind v4 token-mapped classes.

Note: `--font-size-caption` and `--font-weight-caption`/`--font-weight-label`/`--font-weight-subheading` tokens may exist in the shared token package for historical reasons. On the web layer, treat Label (14px/400) as the canonical small-text role. Do not introduce any size below 14px or weight between 400 and 700.

**No glassmorphism on web** (per D-06). Depth is expressed via elevation shadows (`--elevation-low/medium/high`) and `--color-border` strokes instead of backdrop blur.

## Shimmer Animation Spec

CSS keyframe implementation (no Reanimated on web). Per D-02 and Claude's Discretion grant.

```css
@keyframes shimmer {
  0%   { background-position: -200% center; }
  100% { background-position: 200% center; }
}

.shimmer {
  background: linear-gradient(
    90deg,
    var(--color-surface-hover) 25%,
    var(--color-surface-light) 50%,
    var(--color-surface-hover) 75%
  );
  background-size: 200% 100%;
  animation: shimmer 1.5s ease-in-out infinite;
}
```

- Gradient angle: 90deg (left-to-right sweep)
- Duration: 1500ms
- Easing: ease-in-out (not `--motion-easing-standard` — CSS shimmer looks better with symmetric ease)
- Base color: `--color-surface-hover` (#2a2a45)
- Highlight color: `--color-surface-light` (#252540)

Skeleton shaped variants:
- `circle`: `border-radius: 9999px`; use for avatar placeholders
- `text`: height 14px, border-radius 4px; use for text line placeholders
- `card`: border-radius `--border-radius-md`; use for card block placeholders

## Toast Positioning

Per Claude's Discretion grant. Position: **top-right**, 16px inset from viewport edge.

Rationale: Sidebar is on the left, content expands rightward — top-right avoids sidebar overlap and matches standard web conventions for notification placement. Sidebar notification badge (red dot on Proposals) is left-side; toast should not compete with it spatially.

| Property | Value |
|----------|-------|
| Position | `fixed top-4 right-4` |
| Z-index | `z-[100]` (above Modal's z-50) |
| Max toasts visible | 4 |
| Auto-dismiss | 4000ms |
| Progress bar | Animated width from 100% to 0% over 4000ms |
| Enter animation | slide-in from right + opacity 0→1, `--motion-duration-fast` (200ms) |
| Exit animation | opacity 1→0 + scale 0.95, `--motion-duration-fast` (200ms) |

Toast variant colors:
- `success`: background `rgba(46, 204, 113, 0.15)`, border `#2ecc71`, icon `CheckCircle` (Lucide)
- `error`: background `rgba(231, 76, 60, 0.15)`, border `#e74c3c`, icon `XCircle` (Lucide)
- `warning`: background `rgba(230, 126, 34, 0.15)`, border `#e67e22`, icon `AlertTriangle` (Lucide)
- `info`: background `rgba(160, 160, 184, 0.1)`, border `--color-on-surface-secondary`, icon `Info` (Lucide)

## Card Elevation Spec

No glassmorphism. Depth via token shadows + border. Per D-06.

| Elevation | CSS Var | Visual Treatment |
|-----------|---------|-----------------|
| `none` | `--elevation-none` | No shadow, border `--color-border` (subtle separation) |
| `low` | `--elevation-low` | `0 1px 2px rgba(0,0,0,0.2)` + border `--color-border` — default card state |
| `medium` | `--elevation-medium` | `0 2px 4px rgba(0,0,0,0.25)` + border `--color-border` — hovered card / modal backdrop |
| `high` | `--elevation-high` | `0 4px 8px rgba(0,0,0,0.3)` + border `--color-border` — active/focused card |

Default card elevation: `low`. Hover transition: `low` → `medium`, `transform: translateY(-2px)`, `--motion-duration-fast` (200ms), easing `--motion-easing-standard`.
